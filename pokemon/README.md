＊図鑑番号を指定すると対応するポケモンを表示する

＊日本語のポケモン名を指定すると英語のポケモン名を表示する

＊こうげきりょくの強い１から３位のポケモンを表示

＊すばやさの遅い１から３位のポケモンを表示

＊消防士向きのポケモンを探す
　図鑑番号を指定して
　水のタイプをもっていて、すばやさが平均以上の
　ポケモンであれば、採用と表示する

＊発電向きのポケモンを探す
　図鑑番号を指定して
　でんきのタイプをもっていて、ぼうぎょりょくが中央値以上の
　ポケモンであれば、採用と表示する
